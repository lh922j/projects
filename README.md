# AI 기반 함포 무기효과 산출방법론 구축

실무에서 진행한 프로젝트로, 함포 사격 데이터를 기반으로 AI 모델을 활용해 무기효과를 정량적으로 산출하는 방법론을 구축합니다. 실제 함포 사격 데이터를 기반으로, CTGAN 데이터 증강과 AutoGluon 앙상블 회귀를 활용해 함형·기상 조건별 전달정확도(DA)를 정량 산출하는 방법론입니다.

## 📁 프로젝트 구조

```text
weapon/
├── preprocessing.py   # 원시 사격 데이터 정제 및 전달정확도 지표 생성
├── main.py            # 데이터 증강 + 군집화 + AutoGluon 회귀 + 전 조건 예측
├── result.py          # 조건별 가중평균 DA 산출 및 최종 결과 저장
└── 시각화.ipynb        # 탐색적 데이터 분석(EDA) 및 시각화
```

## 📌 파일별 설명

### 1. `preprocessing.py` — 원시 데이터 전처리

- **사격 조건 필터링** : 격파사격(RF) + SALVO 방법만 추출
- **오차값 정제** : 편의오차·거리오차의 R/L/HIT/관측불가 등 문자값 → 수치 변환
- **풍향 숫자화** : N/NE/E/SE/S/SW/W/NW → 방위각(degree) 변환
- **이상치 제거** : IQR + 마할라노비스 거리(χ² p<0.01) 이중 기준 적용
- **표준편의 계산** : `지정사거리 / 사격거리 × 편의오차`
- **함형·함포종류별 분리** : DDH·FFG·PCC·PKG(127mm/76mm), PKM(40mm)
- **전달정확도 지표 4종 생성 및 저장**

| 파일명 | 내용 |
|--------|------|
| `MPI_거리_raw.csv` | 사격 세션별 거리오차 평균(MPI) |
| `MPI_편의_raw.csv` | 사격 세션별 편의오차 평균(MPI) |
| `PREC_거리_raw.csv` | 사격 세션별 거리오차 표준편차(PREC) |
| `PREC_편의_raw.csv` | 사격 세션별 편의오차 표준편차(PREC) |

### 3. `result.py` — 최종 전달정확도 산출

- pred CSV 4종 + raw CSV 4종 로드
- **함형 5종 × 풍속 3종(5·8·10) × 파고 3종(0.5·1.0·1.5)** 조건별 반복
- raw 데이터의 전달정확도(da) **최빈값 분포를 가중치**로 활용해 예측값 가중평균 산출
- 파고 1.0m 조건 기준 최종 결과 출력
- `result.csv` 저장

## ⚙️ 실행 순서

```bash
python preprocessing.py   # 원시 데이터 정제 → raw CSV 4종 생성
python main.py            # 데이터 증강·학습·예측 → pred CSV 4종 생성
python result.py          # 조건별 최종 DA 산출 → result.csv 저장
```

> `main.py`는 실행 중 CLI 입력이 필요합니다 (지표 선택 → 데이터 타입 → 평가지표).
